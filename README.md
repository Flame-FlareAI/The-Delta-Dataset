> **O dataset colaborativo e open-source para treinar a próxima geração de modelos de IA.**

O **The Delta Dataset** é um projeto colaborativo onde qualquer pessoa pode contribuir com exemplos de alta qualidade utilizando o formato **Deltaset**.

Nosso objetivo é construir um dataset aberto, confiável e em constante evolução, que possa ser utilizado tanto pelos modelos **Delta** quanto por qualquer outro modelo de Inteligência Artificial.

---

# 🌎 Nossa missão

Treinar modelos de IA exige milhares (ou milhões) de exemplos de qualidade.

Hoje, grande parte dos datasets utilizados por empresas são privados e inacessíveis para a comunidade.

O **The Delta Dataset** nasceu para mudar esse cenário.

Queremos construir um dos maiores datasets colaborativos e open-source do Brasil, permitindo que qualquer pessoa possa participar da criação de modelos de IA melhores.

# 📦 O que é o Deltaset?

O **Deltaset** é um formato criado para tornar datasets de linguagem mais simples de escrever, revisar e converter.

Exemplo:

```text
[SYS] Você é Delta Ultra Mini.
[SEP]

[USR] O que é Python?
[SEP]

[ASS] Python é uma linguagem de programação de alto nível...
[SEP]
```

O formato foi desenvolvido para ser:

* Simples
* Fácil de ler
* Fácil de escrever
* Fácil de validar
* Fácil de converter para outros formatos

# 🔍 Revisão dos Pull Requests

Nenhuma contribuição é adicionada automaticamente.

Todos os Pull Requests passam por revisão manual para manter o dataset seguro, organizado e com alta qualidade.

Assim que um PR é aberto, ele recebe automaticamente a etiqueta:

* 🕒 **Fila de Espera**

Depois disso, nossa equipe poderá alterar o status para uma das seguintes etiquetas:

## 👥 Etiquetas públicas

### 🔎 Em Inspeção

Estamos revisando cuidadosamente a contribuição.

Nessa etapa verificamos:

* qualidade das respostas;
* formatação;
* possíveis alucinações;
* conteúdo inadequado;
* duplicações;
* conformidade com as regras do projeto.

---

### ✅ Será Usado

A contribuição foi aprovada e será integrada ao dataset.

Obrigado por ajudar o projeto! 🎉

---

### ❌ Não Será Usado

A contribuição não será incorporada.

Os motivos podem incluir:

* informações incorretas;
* conteúdo duplicado;
* exemplos de baixa qualidade;
* violação das regras do projeto.

---

### ❓ Dúvida

Precisamos de mais informações antes de decidir.

Nesse caso deixaremos comentários no Pull Request para esclarecer alguma questão.

---

## 🔒 Etiqueta interna

### 🛠 Edição Oficial

Etiqueta utilizada exclusivamente pela equipe da **Flame-FlareAI** para alterações estruturais e manutenção do projeto.

# ⚠️ Regras importantes

Seu Pull Request poderá ser rejeitado caso contenha:

* ❌ Informações falsas ou inventadas
* ❌ Conteúdo +18, violência extrema ou discurso de ódio
* ❌ Apologia ao crime
* ❌ Dados pessoais de terceiros
* ❌ Spam
* ❌ Conteúdo duplicado
* ❌ Exemplos sem valor para treinamento

Nosso objetivo é manter um dataset útil, seguro e confiável.

---

# 🤝 Como contribuir

1. Faça um **Fork** do repositório.
2. Adicione novos exemplos utilizando o formato **Deltaset**.
3. Verifique se tudo está corretamente formatado.
4. Abra um Pull Request.
5. Aguarde a revisão da equipe.

Contribuições pequenas ou grandes são sempre bem-vindas.
