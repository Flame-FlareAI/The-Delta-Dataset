# 🚀 The Delta Dataset

> **O dataset colaborativo e open-source para treinar a próxima geração de modelos de IA.**

O **The Delta Dataset** é um projeto colaborativo onde qualquer pessoa pode contribuir com exemplos de alta qualidade utilizando o formato **Deltaset**.

Acreditamos que o futuro da Inteligência Artificial deve ser construído pela comunidade, e não apenas por grandes empresas. Nosso objetivo é criar um dataset aberto, confiável e em constante evolução, que possa ser utilizado tanto pelos modelos **Delta** quanto por qualquer outro modelo de IA.

---

# 🌎 Nossa missão

Treinar modelos de IA exige milhares de exemplos de qualidade.

Infelizmente, a maioria dos grandes datasets utilizados atualmente são privados.

O **The Delta Dataset** nasceu para mudar isso.

Queremos construir um dos maiores datasets colaborativos e open-source do Brasil, permitindo que qualquer pessoa possa contribuir para o desenvolvimento de modelos de IA melhores.

# 📦 O que é o Deltaset?

O **Deltaset** é um formato de dataset criado para facilitar o treinamento de modelos de linguagem.

Exemplo:

```text
[SYS] Você é Delta Ultra Mini.
[SEP]

[USR] O que é Python?
[SEP]

[ASS] Python é uma linguagem de programação de alto nível...
[SEP]
````

O formato foi desenvolvido para ser:

* Simples
* Fácil de ler
* Fácil de escrever
* Fácil de validar
* Fácil de converter para outros formatos

# ✅ Revisão das contribuições

Apesar de qualquer pessoa poder contribuir, **nenhuma alteração é adicionada automaticamente ao dataset**.

Todos os **Pull Requests** passam por uma revisão antes de serem aceitos.

Esse processo garante que o projeto continue sendo:

* ✅ Seguro
* ✅ Organizado
* ✅ Bem formatado
* ✅ Livre de spam
* ✅ Livre de conteúdo malicioso ou inadequado
* ✅ Com exemplos de alta qualidade

Nosso objetivo é manter um dataset que qualquer desenvolvedor possa utilizar com confiança.

---

# 🤝 Como contribuir

1. Faça um **Fork** do repositório.
2. Adicione novos exemplos utilizando o formato **Deltaset**.
3. Verifique se tudo está corretamente formatado.
4. Abra um **Pull Request**.
5. Aguarde a revisão da equipe.

Contribuições de qualquer tamanho são bem-vindas!
